# 用户使用说明

这是一款基于flutter开发的简易学习辅助工具，纯粹的绿色软件，您无需担心个人数据安全问题，希望它能改善您的学习体验。

我希望这款软件使您的学习更有成就感。

## 软件说明： 

### 1-程序配置文件 

配置文件夹路径：C:\Users\你的用户名\Documents\studyclock

程序日志路径：C:\Users\你的用户名\Documents\studyclock\Log.txt 

学习日志路径：C:\Users\你的用户名\Documents\studyclock\StudyClockLogs.txt 

学科文件路径：C:\Users\你的用户名\Documents\studyclock\StudyClockSubjects.json 

统计文件路径：C:\Users\你的用户名\Documents\studyclock\StudyClockSubjectsStats.json

假如存在数据差异，您可以在这中间进行查看修改。 请谨慎对待这四份数据，如果彻底删除将无法找回！

### 2-软件介绍 

（1）关于快捷方式创建： 碍于开发者目前没有掌握进一步的安装包制作，因此暂不支持桌面快捷方式创建功能，当前安装包的使用方式为：选择安装路径（可选，默认为当前文件夹下）-> 找到my_study_clock文件夹 -> 点击进入，点击release文件夹 -> 右键my_study_clock.exe -> 选择“固定到任务栏”或“固定到‘开始’”或“创建快捷方式” 

（2）关于计时器： 在倒计时（选择学习时长）设置好后，您可以直接点击倒计时显示区下方的“开始”启动倒计时并触发倒计时设置的折叠动作。 您也可以直接点击“开始”按键进行正向计时。 

（3）关于学科栏： 您可以点击科目条目以选中，再次点击可取消选中状态。 

（4）关于时长统计： 您的学习时长被保存在统计文件当中 （C:\Users\你的用户名\Documents\studyclock\StudyClockSubjectsStats.json） 在软件中删除学科不会将您的学习时长删去，同理，如果您希望删除学习统计时长，可以进入该文件中进行编辑。 如果更换设备，且您不想丢失当前数据，请注意做好备份，并按照 C:\Users\你的用户名\Documents\studyclockstudyclock--Log.txt | --StudyClockLogs.txt | --StudyClockSubjects.json | --StudyClockSubjectsStats.json的格式进行备份，即，确保将上述四个文件转存至Documents文件夹下的studyclock文件夹中 

（5）关于该软件当前版本的总体概述： 当前软件没有实现进一步的统计功能，也尚不支持在线功能，但在后续版本中将逐步更新；我为您使用软件快捷方式创建以及备份带来的不方便致以歉意。
